# Rails7_1_3sample

| Dependency | Version |
|:-----------|:--------|
| Ruby       | 3.3.0 |
| Rails      | 7.1.3.2 |

## Getting Started

1.  Clone the project

  `git clone git@github.com:EGiataganas/rails7_1_3.git`


2. Install the project's dependencies using bundler

  `bundle install`

## Server

  `rails s`
